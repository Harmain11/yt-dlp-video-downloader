# YT-DLP Video Downloader

## Overview  
This notebook demonstrates how to download videos from a web page using the `yt_dlp` Python library. It provides a simple example script to extract and save the best quality video using yt_dlp.

## Features  
- Download videos from URLs  
- Save videos using the video title as the filename  
- Select the best available video quality automatically  

## Tech Stack  
- Python  
- Jupyter Notebook / Google Colab  
- yt_dlp Python library  

## How to Use  
1. Clone or download this repository.  
2. Open the notebook `yt-dlp-video-downloader.ipynb` in Google Colab or Jupyter Notebook.  
3. Update the URL variable with the desired video page URL.  
4. Run the code cells to install dependencies and download the video.
